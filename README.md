# 🖼️ Image Captioning Using CLIP

## 📌 Overview

This project demonstrates an image captioning system powered by the CLIP (Contrastive Language–Image Pretraining) model. The application analyzes an input image and predicts the most relevant caption by measuring the semantic similarity between image and text embeddings.

The project showcases the practical use of multimodal artificial intelligence, where both images and text are represented in the same embedding space for accurate matching.

---

## 🚀 Features

- Upload and process images
- Generate image embeddings using the CLIP model
- Compare image embeddings with multiple text captions
- Predict the most relevant caption
- Display the image along with the predicted result
- Simple and beginner-friendly implementation

## ▶️ How It Works

1. Load the pre-trained CLIP model.
2. Read the input image.
3. Convert the image into an embedding.
4. Convert candidate captions into text embeddings.
5. Calculate similarity scores.
6. Select the caption with the highest similarity.
7. Display the final prediction.
